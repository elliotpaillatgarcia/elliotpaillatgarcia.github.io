Présentation en anglais d'un territoire économique et culturel

Ce projet consiste à présenter un territoire ayant accueillis les Jeux Olympiques avec deux langues le Francais qui présente l'aspect historique archituctural ou encore l'impact sur la ville et l'Anglais qui présente l'aspect géographique, humain et économique. Ce projet fut un travail de groupe de trois où nous avons dù choisir une ville pour notre part ce fut athènes car elle avait déjà un passé avec les Jeux et est une ville qui a une histoire très dévéloppés de part la mythologie et donc les monuments présents.

Applications utilisés:

Word, Power Point ainsi que de nombreux sites internets

Apprentissages:

Recherche et récupération des informations importantes ainsi qu'une analyse et vérification des données obtenus suivi d'une présentaion de nos résultats avec nos découvertes de la ville.

Régression sur des données réelles

Ce projet consiste à faire un liens entre de nombreuses variables afin de pouvoir déterminer le prix de nombreux logements. Ce projet est fait en binomes et avait pour but de trouver des prix de logements avec le plus de precisions possible et pour cela nous avons dû trouver un coefficiant de corrélation qui était suffisant en utilisant de nombreux calculs et de nombreuses variables de types variées ce qui fut compliqué, fastidieux et long.

Applications utilisés:

Excel et R

Apprentissages:

Apprentissage du code R ainsi que la création entre de nombreuses variables

Conception et implémentation d'une base de données

Ce projet consiste à créer une base de données et la remplir grâce a une application sous python pour aider des auniers à mieux organiser les productions ainsi que les venttes. Ce projet était en binomes donc nous avons commencés par s'approprié le sujet ensuite nous avons réparti le travail en fonction des compétences de chacun puis nous avons avancé sur notre travail individuel pour ensuite faire le plus gros du travail qui consiste a automatiser la base de données avec le code python en passant par des fichiers CSV et de manières simplifié pour utilisation possible par n'importe qui.

Applications utilisés:

Python et easy PHP

Apprentissages:

Apprendre a créer une base de données, faire une application via python et connécter un code python à une base de données.
